### Technologies

Vue, Vuex, Vue-Router, Axios, Bootstrap, uuid, Tmdb Api

### Features

- Listing movies according to their features
- Movie search
- Viewing movie detail
- Watching the movie trailer
- Saving and removing movies to watch list for logged in users
